## Web 3.0 성적 관리 시스템

---

## 📌 프로젝트 개요
**Web 3.0 성적 관리 시스템**은 블록체인 기술을 활용하여 학생들의 성적 및 출석 정보를 관리하는 탈중앙화된 시스템입니다. 기존의 중앙 서버 기반 성적 관리 시스템과 달리, **SoulBound Token(SBT)** 을 활용하여 성적 데이터를 개인화하고 블록체인에 저장함으로써 데이터 무결성과 보안성을 보장합니다.

### 🔥 주요 목표
- **Web 3.0과 SBT를 활용한 데이터 무결성 보장**
- **학생이 자신의 성적 및 출석 정보를 직접 소유하고 관리**
- **블록체인 및 분산형 저장 시스템(IPFS) 기반 데이터 보호**
- **학습 동기 부여를 위한 게이밍 요소 도입**
- **Web UI를 통해 사용자 친화적인 환경 제공**

---

## 🔗 기술적 배경

### 🏗️ 핵심 기술
- **📌 블록체인 (Blockchain)** : 데이터 신뢰성과 무결성을 보장하는 분산 원장 기술 활용
- **📝 스마트 컨트랙트 (Smart Contract)** : ERC-721 기반의 SBT(SoulBound Token) 발행
- **📂 IPFS (InterPlanetary File System)** : 성적 및 출석 데이터를 저장하고 해시값을 블록체인에 저장하여 효율적 관리
- **🖥️ Web 3.0 기술** : 탈중앙화된 성적 관리 시스템 구축
- **🌐 프론트엔드** : React.js, JavaScript
- **⚡ 백엔드** : Node.js, Express.js
- **📊 데이터베이스** : MySQL
- **🔐 인증 시스템** : Google Login API

## ⚙️ 시스템 아키텍처

### 💡 전체 시스템 개요
1. **프라이빗 블록체인 네트워크 구축**
   - AWS EC2를 이용하여 가상 서버 환경 구성
   - Ganache CLI를 활용하여 스마트 컨트랙트 배포 및 테스트

2. **SBT(SoulBound Token) 발행**
   - 성적 및 출석 정보를 SBT 형태로 발행하여 학생들에게 부여
   - NFT의 거래 가능성을 차단하는 **Locked 이벤트** 구현

3. **IPFS 기반 데이터 저장**
   - 성적 및 출석 데이터를 직접 블록체인에 저장하지 않고, **IPFS에 저장한 후 해시값만 블록체인에 저장**
   - Gas Fee 절감 및 데이터 무결성 보장

4. **웹 애플리케이션 구축**
   - **React.js** 기반의 사용자 친화적 UI 설계
   - **Node.js & Express.js** 기반의 백엔드 구축
   - **Google Login API** 활용하여 인증 시스템 구현

## 🛠️ 개발 및 실행 방법

### 1️⃣ 프로젝트 클론
```bash
git clone https://github.com/20Yiju/Capston_web3.0GMS.git
cd Capston_web3.0GMS
```

### 2️⃣ 환경 설정
```bash
npm install
```

### 3️⃣ 블록체인 네트워크 실행
```bash
ganache-cli --deterministic --host 0.0.0.0
```

### 4️⃣ 스마트 컨트랙트 배포
```bash
truffle migrate --reset
```

### 5️⃣ 데이터베이스 서버 실행
```bash
node index.js
```

### 6️⃣ 웹 서비스 실행
```bash
npm start
```

### 7️⃣ 브라우저에서 실행
```bash
https://tomato.ne.kr:4000/
```
