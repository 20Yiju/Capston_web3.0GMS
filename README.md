## Web 3.0 성적 관리 시스템

---

## 0. Executive Summary

### 0.1. 배경

대한민국의 교육 환경에서 학생들의 학업 성취를 효과적으로 관리하는 것은 매우 중요합니다.

본 프로젝트는 **SoulBound Token(SBT)**을 활용하여 학생들에게 거래 불가능한 토큰 형태로 성적 및 출석 정보를 제공하여 학기 중 성취도를 직관적으로 파악할 수 있게 합니다.

이와 함께 게이밍 요소를 결합하여 학생들의 학습 동기를 향상시키고, 블록체인과 분산 저장 시스템을 이용하여 보안성을 강화하여 더욱 신뢰할 수 있는 학습 환경을 조성하고자 합니다.

### 0.2. 주제

- **프라이빗 블록체인 네트워크**에서 스마트 컨트랙트를 실행하고,
- 학생별 **고유한 성적 토큰(SBT)**을 발행하여 **안전하고 투명한 성적 관리 시스템을 구축**하는 것이 목표입니다.

## 📌 1. 프로젝트 개요

**Web 3.0 성적 관리 시스템**은 블록체인 기반 성적 및 출석 관리 시스템입니다.

기존의 중앙 서버 기반 시스템과 달리, 탈중앙화된 구조를 적용하여 학생이 자신의 학습 데이터를 직접 관리할 수 있도록 합니다.

### 🔥 1.1 주요 목표

- **Web 3.0과 SBT를 활용한 데이터 무결성 보장**
- **학생이 자신의 성적 및 출석 정보를 직접 소유하고 관리**
- **블록체인 및 IPFS 기반 데이터 보호**
- **학습 동기 부여를 위한 게이밍 요소 도입**
- **Web UI를 통해 사용자 친화적인 환경 제공**

---

## 🔗 2. 기술적 배경

### 🏗️ 2.1 주요 기술

- **📌 블록체인 (Blockchain)**: 데이터 신뢰성과 무결성을 보장하는 분산 원장 기술 활용
- **📝 스마트 컨트랙트 (Smart Contract)**: ERC-721 기반의 SBT(SoulBound Token) 발행
- **📂 IPFS (InterPlanetary File System)**: 성적 및 출석 데이터를 저장하고 해시값을 블록체인에 저장하여 효율적 관리
- **🖥️ Web 3.0 기술**: 탈중앙화된 성적 관리 시스템 구축
- **🌐 프론트엔드**: React.js, JavaScript
- **⚡ 백엔드**: Node.js, Express.js
- **📊 데이터베이스**: MySQL
- **🔐 인증 시스템**: Google Login API

## 4. 구현 및 평가

### 4.1 구현

### 4.1.1 구현에 사용된 개발 도구

- **Ganache-cli**: 테스트 목적으로 사용할 수 있는 가상의 이더리움 네트워크.
- **Truffle**: 이더리움 기반의 디앱을 개발할 수 있게 도와주는 블록체인 프레임워크.
- **Remix**: 스마트 컨트랙트 코드를 작성하고 컴파일할 수 있는 브라우저용 IDE.
- **Metamask**: 가상자산을 웹브라우저와 모바일 장치에서 이더리움 블록체인과 상호작용할 수 있는 지갑.
- **OpeanSea**: NFT 거래소. 특정 계정에게 전송된 토큰을 열람하는데 사용.
- **Infura**: 블록체인 노드를 제공하는 서비스. 토큰에 들어갈 내용을 메타데이터의 형태로 업로드하고 결과로 URI를 반환받기 위해 사용.
- **IPFS**: 분산된 파일 시스템에 데이터를 저장하고 공유하는 P2P 네트워크.
- **VScode**: 소스코드 편집기.
- **Solidity**: 블록체인에서 거래를 자동화하는 스마트 컨트랙트 코드를 작성하는데 사용되는 프로그래밍 언어.
- **Javascript**: 객체 기반의 스크립트 프로그래밍 언어.
- **React**: 사용자 인터페이스를 만들기 위한 자바스크립트 라이브러리.
- **Google Login API**: 구글 로그인을 사용하기 위해 사용.
- **MySQL**: 관계형 데이터베이스 관리시스템.

### 4.1.2 구현 환경

- **AWS EC2**: 아마존에서 제공하는 클라우드 컴퓨팅 서비스. 가상의 서버를 임대해주는 서비스.

### 4.2 실험 설정

### 4.2.1 웹서비스 실행

웹서비스는 **AWS EC2 인스턴스(Ubuntu 환경)**에서 실행됩니다.

1. **AWS EC2 인스턴스 생성 및 시작**
2. **원격 저장소 복제**
    
    ```bash
    git clone <https://github.com/20Yiju/Capston_web3.0GMS>
    ```
    
3. **블록체인 네트워크 실행**
    
    ```bash
    ganache-cli --deterministic --host 0.0.0.0
    ```
    
4. **스마트 컨트랙트 컴파일 및 배포**
    
    ```bash
    truffle deploy or truffle migrate
    ```
    
5. **데이터베이스 서버 실행**
    
    ```bash
    node index.js
    ```
    
6. **웹 서비스 실행**
    
    ```bash
    npm start
    ```
    
7. **브라우저에서 실행**https://tomato.ne.kr:4000/
8. **교수님 페이지에서 학생에게 토큰 발행**
9. **학생 페이지에서 발행된 토큰 확인**
